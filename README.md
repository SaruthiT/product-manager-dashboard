# Product Manager Dashboard

A prototype dashboard built with **Cloudflare Workers** and **D1 Database** to analyze customer feedback.

## Features
- Displays total, positive, neutral, and negative feedback metrics.
- Shows sentiment distribution charts.
- Lists actionable insights and key themes from feedback.
- Filter by sentiment and sort feedback by date or user.

## Tech Stack
- **Frontend/Backend**: Cloudflare Worker (JavaScript)
- **Database**: D1 (Cloudflare)
- **Deployment**: Cloudflare Workers
- **Other tools**: Wrangler CLI
